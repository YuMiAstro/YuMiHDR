YuMiHDR — PixInsight update repository

This is the update repository for the YuMiHDR script module for PixInsight. To install or update from inside PixInsight:

  1. Open RESOURCES → Updates → Manage Repositories.
  2. Click Add and paste the following URL:

https://yumiastro.github.io/YuMiHDR/

  1. Click OK, then RESOURCES → Updates → Check for Updates.
  2. Apply the proposed update and restart PixInsight when prompted.
  3. After restart, run SCRIPT → YuMiHDR → YuMiHDR.

What's inside

  • updates.xri — the PixInsight update manifest read by the updater.
  • YuMiHDR-script-1.0.0.zip — the actual script payload mirrored against the PixInsight installation directory layout.

Source code, documentation and issue tracker are at github.com/YuMiAstro/YuMiHDR.
